Arduino House

Project Objective

Our objective is to create functioning house using buttons to control internal lighting and a security system composed of a sensor and a buzzer.

Basic Wiring

As the project required the use of multiple LEDs and electronic devices, a basic understanding of how some of these devices need to be wired would be beneficial.

For example above is the circuit layout to power a single LED which required a resistor and LED in series connected to the power source which in this case was the Arduino.

House Wiring

Arduino House Schematic

By repeating the circuit layout for a single LED (as shown previously), we could repeat that design 4 times to light up 4 rooms in the house with every light attached to a button which would act as a light switch.

Alongside the lighting we needed to wire the security system which composed of a ultrasonic sensor to detect anyone coming into the house and a buzzer to sound when a person is detected by the ultrasonic sensor.

Arduino House Wiring

The result of digital schematic is as shown above.

The left side contains the button controls to the LEDs in each room (1 off and 1 on per light) and on the right hand side is the Arduino controller to manage and power the entire circuit

Final Product

House Exterior

The outer design of the Arduino House

The outer design is double layered such that the wiring for LEDs to each floor is hidden as it would be in a real house.

House Interior

The inside of the Arduino House

The Basement level contains the Arduino unit and controls to every light as well as the buzzer in the event the ultrasonic sensor detects someone coming through the front door.

As mentioned earlier the main door has the ultrasonic sensor above it which would act as a security system and each room in the house has an LED to light them up.
